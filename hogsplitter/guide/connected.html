Hog Splitter Pro

Connected Mode — multi-device timing

Overview Standalone guide

Experimental: Connected Mode is very experimental. Multi-unit radio timing, discovery, and Chronocurl BLE integration are still being hardened; you may see crashes, dropped links, missed or duplicated splits, or rough edges in the UI. For anything where timing must be dependable, use standalone mode (main user guide) unless you are deliberately testing this stack.

What Connected Mode is

Connected Mode links multiple Hog Splitter Pro units (and optional Chronocurl hardware) so splits can be shared over radio. One device enters Connected Mode at boot; after discovery, the main screen shows a NET: line and a multi-column history grid. Treat everything below as describing work-in-progress behavior that may change between firmware releases.

  • Columns: Each connected peer’s split times appear in a dedicated column on the Running screen—your unit’s column plus one column per remote Hog Splitter Pro.
  • Hog Splitter Pro limit: Up to four Hog Splitter Pros can take part in one connected swarm (this device plus up to three peers over ESP-NOW).

Standalone timing (single device, no radio grid) is documented in the main user guide.

Entering Connected Mode at boot

Power on the unit, then hold the Split button for several seconds until the latch screen appears. Release when prompted. The device stays in Connected Mode for that session.

If the radio stack fails to initialize, the firmware shows an error briefly and falls back to standalone mode.

Discovery phase

After a successful boot into Connected Mode, the device runs a short discovery window. Nearby peers are counted; the screen updates about once per second with remaining time.

Running — main display

After discovery completes, the UI matches the standalone timer style for the large elapsed readout, but adds a peer count and a history grid where each Hog Splitter Pro in the swarm has its own column (local column plus one per remote peer), up to four units total.

  • Split button: same as standalone — records a split.
  • Main timer precision and History precision (Settings) apply the same way as standalone: the large readout uses Main timer precision; all history cells use History precision.
  • Intervals above 20 seconds are not added to the on-device history in Connected Mode (same 20 s cap as standalone history).
  • Intervals above 59 seconds still mark the interval as “too long” for the big readout (dashed placeholder matching Main timer precision), same as standalone.

Settings in Connected Mode

Once the device is in the Running phase (not during discovery), you can open settings with Wheel-push from the main screen, same as standalone:

  • Wheel-push: next settings page (with beep).
  • Wheel-up / Wheel-down: adjust the current item (with beep / preview tone).
  • Split button: save and exit settings.

Chronocurl integration

Chronocurl gate hardware pairs with the swarm over Bluetooth LE. Discovery shows how many Chronocurls were seen during the discovery window.

  • Capacity: Up to four Chronocurl devices are supported per swarm.
  • Host stopwatch: One Hog Splitter Pro in the group acts as the BLE host for Chronocurls. When a paired Chronocurl detects the laser beam broken, that host records a split—the same timing action as pressing Split on the stopwatch (including sharing the interval over radio when other Hog Splitter Pros are linked).
  • Laser cooldown: After a Chronocurl-triggered split, the laser on that Chronocurl is held disabled for 5 seconds before it can produce another break event.
